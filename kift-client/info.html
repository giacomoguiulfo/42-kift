K.I.F.T

VUI (Voice User Interface) based on CMUSphinx written in C and JavaScript.

Getting Started

# Clone Repo
	  git clone https://github.com/giacomoguiulfo/kift.git
	  cd kift

	  # Build Dependencies
	  sh build.sh

	  # Run Server
	  ./kift

	  # Launch App
	  electron kift-client/main.js

Built With

  • Electron - Front End
  • CMU SPHINX - Recognizer Library
  • node-record-lpcm16 - Recording Library
